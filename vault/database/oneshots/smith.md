# Reminders
### After Short Rest
- **Expend HP Dice to regain HP**
  Regain *(d8 + CON)* HP per expended die.

### After Long Rest
- **Regain all HP**
- **Regain [*Level / 2 (round down)*] HP Dice**
- **Regain all Spell Slots**

### After Level-Up
- **Recalculate HP**
- **Add 1 HP Die**
- **Check Class Level-Up**
